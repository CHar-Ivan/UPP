# WPF Приложение "Система управления библиотекой"

## Описание проекта

Это полнофункциональное WPF приложение для управления библиотекой с использованием **SQL Server**. Приложение работает с локальным экземпляром SQL Server (LocalDB).

```
┌─────────────┐          ┌─────────┐
│   Authors   │ 1───∞───→│  Books  │
│ (5 полей)   │          │(6 полей)│
└─────────────┘          └────┬────┘
                              │
                         ∞    │
                          ┌───┴──────────┐
                          │ Borrowings   │

                          └─────────────┘
```

## 🚀 Запуск приложения

### Способ 1: Запуск через Visual Studio

1. Откройте проект в Visual Studio 2022+
2. Убедитесь что установлен SQL Server Express или LocalDB
3. Дождитесь завершения восстановления пакетов NuGet
4. Нажмите **F5** или кнопку "Start"
5. Приложение автоматически создаст БД на SQL Server с примерами данных

### Способ 2: Запуск через командную строку

```bash
cd "C:\Users\asus\source\repos\лаб29.1"
dotnet run
```

### Способ 3: Публикация и запуск .exe

```bash
dotnet publish -c Release
# Затем запустите полученный .exe файл из папки bin\Release\net8.0-windows\publish\
```

## 🐛 Решение проблем

### Проблема: "БД не создается"
**Решение**: Убедитесь, что папка `C:\Users\[ВашЕмя]\AppData\Local\` существует и доступна для записи.

### Проблема: "Долгая первая сборка"
**Решение**: Это нормально - первая загрузка Entity Framework может занять 2-5 минут. Последующие сборки будут быстрее.

### Проблема: "Не загружаются данные при выборе"
**Решение**: Нажмите кнопку "Обновить" на нужной вкладке.
