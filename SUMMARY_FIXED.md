### Режимы работы

#### 1. Offline режим (по умолчанию)
- Приложение стартует с sample данными
- Данные хранятся в памяти
- Позволяет добавлять/удалять записи
- Данные теряются при закрытии приложения

#### 2. Online режим (SQL Server)
- Нажмите кнопку "🔗 Подключиться и синхронизировать"
- Все данные синхронизируются с SQL Server
- Изменения сохраняются в базе данных

## Известные особенности

### Offline режим
✅ **Плюсы:**
- Работает без интернета
- Быстрый доступ к данным (в памяти)
- Не требует установки SQL Server

⚠️ **Минусы:**
- Данные теряются при закрытии приложения
- Нет синхронизации между несколькими компьютерами

### SQL Server режим
✅ **Плюсы:**
- Данные сохраняются в базе данных
- Синхронизация данных
- Многопользовательский доступ

⚠️ **Минусы:**
- Требует установку SQL Server
- Требует подключение к сети
